🤖 Настройка бота

1

Создайте бота в @BotFather

Откройте Telegram → найдите @BotFather → отправьте /newbot → введите имя и username (должен заканчиваться на _bot) → скопируйте токен.

2

Узнайте ваш Chat ID

Откройте Telegram → найдите @userinfobot → отправьте любое сообщение → скопируйте ваш ID (число).

3

Вставьте данные ниже

После сохранения бот автоматически запустится и начнёт принимать заказы.
